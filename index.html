Technical Logic — Confluence Intelligence Engine

1) Momentum Confluence — 45 pts

  • EMA stack: EMA20 > EMA50 > EMA200 confirms trend structure.
  • Bollinger squeeze: narrow band width plus price above mid-band indicates compression breakout potential.
  • Relative strength: asset 30D return minus BTC 30D return highlights outperformers.
  • RSI health: rewards 45–70 range and flags overheated readings.

2) On-chain / Liquidity — 25 pts

  • TVL size and 1D TVL direction via DeFiLlama protocol mapping.
  • Exchange net flows and funding rates are adapter-ready fields; this static version marks unavailable data transparently.
  • Liquidity gap/FVG logic is defined as a future OHLC structure module for impulse candles and gap retests.

3) Sentiment Velocity — 20 pts

  • CoinGecko sentiment vote ratio contributes when available.
  • Latest catalyst links open an asset-specific Google News query.
  • Social volume/NLP spike scoring is adapter-ready and shown as neutral until a social/news API is connected.

5 Power Rules

Rule 1 — Momentum Trigger

If EMA20 > EMA50 > EMA200 and RS vs BTC > 0, trigger Strong Momentum candidate.

Rule 2 — Squeeze Breakout

If Bollinger bandwidth is compressed and price closes above mid-band, add breakout confluence.

Rule 3 — Anti-FOMO Guard

If RSI is extreme and funding is extreme, force Overextended status regardless of price action.

Rule 4 — TVL Confirmation

If price momentum is positive and TVL 1D change is positive, increase confidence score.

Rule 5 — Laggard Rotation

If a sector has high average score and a coin has low 24H movement but positive sector correlation, flag as Laggard Watch.
